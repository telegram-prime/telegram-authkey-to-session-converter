# HC2S Converter – Telegram **AuthKey / HEX:DC → SESSION+JSON or TDATA**

> A one‑stop tool that turns any Telegram key (AuthKey or HEX:DC) into a *Session+JSON* file for Telethon or into a *TDATA* format.

## ⚙️ Core Features

| Feature | Description |
| --------|-------------|
| **All key types accepted** | AuthKey, HEX:DC, DC:HEX |
| **Output formats** | Telethon *Session+JSON* or *TDATA* |
| **Multithreading** | Parallel processing of multiple files for faster bulk conversion. |
| **Flexible session handling** | Keep current session or log in to a new one |
| **Cold Conversion** | Option to convert SESSION+JSON without connecting to the account |
| **Old Layer Compatibility** | Option to set compatibility with old Telegram layer or save it as a new layer |
| **Session management** | Optionally disable all other active sessions during conversion |
| **Connection tuning** | Custom retry count & delay |
| **2FA automation** | Manual single code for all accounts |
| **Device emulation** | Android, iOS, desktop – old & new sessions |
| **Proxy support** | HTTP/SOCKS5, multi‑line list; each connection gets its own proxy |
| **Result sorting** | Valid / invalid / timed‑out / bad‑2FA → separate folders |
| **Source backup** | Original files copied before operation |
| **Full logging** | “Program Console Log” window tracks progress & errors |
